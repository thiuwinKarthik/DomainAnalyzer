# Quick Start Guide

## Prerequisites

1. **Node.js** (v14 or higher)
2. **Backend running** on `http://localhost:8000`

## Installation & Running

1. **Navigate to frontend directory:**
```bash
cd frontend
```

2. **Install dependencies:**
```bash
npm install
```

3. **Start the development server:**
```bash
npm start
```

4. **Open your browser:**
The app will automatically open at `http://localhost:3000`

## Usage

1. Enter a domain name (e.g., `airbnb.com`, `spotify.com`)
2. Click "Analyze" button
3. View company details and graph visualization
4. Download company information using the download buttons

## Features

✅ Domain input with validation
✅ Complete company details display
✅ Interactive knowledge graph visualization
✅ Download as JSON or CSV
✅ Responsive design
✅ Custom CSS styling

## Troubleshooting

- **Backend not responding**: Make sure backend is running on port 8000
- **Graph not showing**: Check browser console for errors
- **CORS errors**: Ensure backend CORS is configured correctly
